## Сборка многомодульных программ. Вычисление корней уравнений и определенных интегралов

## Постановка задачи

С заданной точностью ε вычислить площадь плоской фигуры, ограниченной тремя кривыми.

## Вариант задания

1. **Уравнения кривых:** вариант I.11 (повышенной сложности на +3 задачи)

    Плоская фигура задается во время сборки программы в виде текстового описания 

2. **Методы приближенного решения уравнений:** вариант II.3 - Метод касательных (Ньютона).

3. **Квадратурные формулы:** вариант III.1 - формула прямоугольников.
